# cn-portal-notes

一个用于归档和发布独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为通用 HTML 文档的存储与展示空间。

## 项目简介

本仓库主要存放各类独立的 HTML 页面，这些页面可能包含信息展示、工具页面、静态文档或其他前端内容。所有页面均为独立文件，不依赖后端服务，可直接通过浏览器打开或部署到静态托管平台。

## 目录说明

```
.
├── README.md          # 本文件
├── index.html         # 仓库首页/导航页（如有）
└── pages/             # 存放独立 HTML 页面的目录
    ├── example1.html
    ├── example2.html
    └── ...
```

- `pages/`：主要的页面归档目录，每个 HTML 文件代表一个独立的页面。
- 根目录下的 `index.html` 可作为仓库的简易导航或入口页面（可选）。

## 页面归档说明

- 所有页面均为纯 HTML/CSS/JavaScript 实现，不包含服务器端代码。
- 页面内容可能涉及信息整理、工具功能、学习笔记或静态展示等。
- 每个页面独立运行，互不依赖。
- 归档的页面可能不定期更新、补充或整理。

## 使用方式

1. 直接克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可将仓库部署到任何支持静态文件的托管服务（如 GitHub Pages、Netlify 等），实现线上访问。
